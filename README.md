# TechMark 教育沙盘模拟

基于 TechMark 框架的教育企业商业模拟经营游戏。

## 游戏规则概览

### 基础设置
- **玩家**: 2-6家教育企业，共享同一市场
- **轮数**: 5轮
- **初始资金**: 1,000万元
- **负债利率**: 8%/轮

### 市场与产品
| 市场 | 学员上限 | 学员预算上限 |
|------|---------|------------|
| T1 (一线) | 10,000 | 10,000 |
| T2 (二线) | 30,000 | 4,000 |
| T3 (三线) | 100,000 | 2,000 |

**产品矩阵**: 3市场 × 3产品 = **9个产品**
- 1V1: 教师配比 1:20，默认新增100人/轮
- Class: 教师配比 1:10，默认新增300人/轮
- APP: 无需教师，默认新增1000人/轮

### 每轮决策
为每个产品决定：
1. **price** - 产品单价
2. **teachers** - 分配教师人数
3. **marketing_budget** - 市场投入预算（用于获客）

### 品牌效应
按该市场3个产品质量分总和排名：
- 第1名: 获客成本降低20%
- 第2名: 获客成本降低10%

### 退费规则
- **师资不足**: 超出教师服务能力的学员退费
- **预算超限**: 单价超过学员预算则全部退费
- **退费惩罚**: 该产品下轮质量分降低20%

### 财务模型
- 收入 = 学员数 × 单价
- 教师成本 = 教师数 × 1万/轮
- 获客成本 = 学员数 × 单位获客成本 × (1-折扣)
- 负债利息 = 负债 × 8%/轮

### 胜利条件
5轮后**净资产**（现金 - 负债）最高者获胜。

## 快速开始

### 1. 初始化游戏

```bash
cd techmark_edu
python main.py --init
```

输入玩家数量和公司名称，系统会生成初始状态和第1轮决策模板。

### 2. 填写决策

编辑 `data/decisions_round_1.csv`：

```csv
company_name,product_id,price,teachers,marketing_budget
公司A,T1_1V1,8000,5,100000
公司A,T1_Class,3000,10,80000
...
```

### 3. 运行单轮

```bash
python main.py --round 1
```

系统会：
- 读取决策文件
- 计算学员分配、财务结果
- 输出结果到 `data/results/`
- 生成下一轮决策模板

### 4. 运行完整游戏

如果已填写所有5轮决策：

```bash
python main.py --play-all
```

### 5. 查看规则

```bash
python main.py --rules
```

## 策略提示

1. **价格策略**: 定价需在学员预算范围内，过高会导致全部退费
2. **师资配置**: 教师不足会导致学员退费，但教师过多会增加成本
3. **市场投入**: 投入获客预算可获得更多学员，但需考虑获客成本折扣
4. **质量分维护**: 避免退费，否则质量分下降会影响后续获客
5. **资金管理**: 注意负债利息，避免资金链断裂

## 进阶：调整游戏参数

编辑 `config.py` 可修改：
- 初始资金、轮数
- 市场容量、学员预算
- 获客成本、教师工资
- 品牌折扣比例
- 质量分惩罚力度
